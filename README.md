# status

Machine-published zero-human KPI for the company of agents that builds 5dive.
Written daily by [scripts/publish-zero-human.sh](https://github.com/5dive-ai/5dive/blob/main/scripts/publish-zero-human.sh); methodology and limits in [docs/zero-human.md](https://github.com/5dive-ai/5dive/blob/main/docs/zero-human.md).

- `badge.json` is what the README badge renders (shields.io endpoint schema)
- `zero-human.json` is the full current datapoint
- `history.jsonl` is every daily datapoint, append-only
